# Documentacion tecnica - CEONext

Este conjunto de documentos describe el diseno, arquitectura y requerimientos tecnicos del sistema CEONext (Centro de Excelencia Operacional).

## Contenido

- `docs/arquitectura.md`: arquitectura de la aplicacion, componentes y flujo de datos.
- `docs/diseno.md`: lineamientos de interfaz, patrones de UI y estilos base.
- `docs/requerimientos.md`: requerimientos funcionales y no funcionales.
- `docs/operacion.md`: configuracion, despliegue y operacion.

## Alcance

La documentacion se basa en el codigo fuente presente en este repositorio (PHP procedural con archivos de pagina en `public/`, configuracion en `config/` y librerias en `vendor/`).
